# The Flywheel Gusto Already Owns

Work sample for Gusto's PM Internship (Summer 2026) by [Sidharth Sundaram]

**Live:** [gusto-pulse.vercel.app](https://gusto-pulse.vercel.app) 

## What this is

An interactive strategy artifact arguing that Gusto's most defensible asset isn't its HR feature set — it's the verified financial signal generated every time it processes payroll. The artifact maps the thesis, proposes a three-phase validation plan (not a feature), and names where the thesis breaks.

Three tabs:
- **The Thesis** — Why payroll data is structurally different from any other financial signal, what Gusto Money already does, and an animated demo of the minimum viable test (a proactive email alert)
- **How to Test It** — Segmentation hypothesis, adapted Sean Ellis survey with kill criteria, and an email-based MVP with success metrics
- **Risks + Why Sid** — Honest competitive positioning (including where Gusto loses), six named risks, JD-to-evidence mapping with gaps stated upfront

## What this isn't

A consulting deck. A feature spec handed down from outside. This is one candidate's attempt to demonstrate strategic judgment, PM craft, and intellectual honesty about a product they don't work on — built in a few days with public information.
